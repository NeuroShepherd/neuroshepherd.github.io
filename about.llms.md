# About Me

[ ORCID](https://orcid.org/0000-0003-1769-7580) [ Github](https://github.com/NeuroShepherd)

I’m a statistical programmer and epidemiologist. My work focuses on building tools to simplify and improve the analysis of research data, spreading open science practices, and improving reproducibility in research.

I tend to do statistics in R, create deployment projects in Python/SQL and distribute with Docker, and I write about statistics and simulations from time to time.

## Tools & Skills

**Languages:** R, Python, SQL, Go, HTML/CSS

**Areas:** Statistical modeling, simulation, data visualization, R package development, web scraping
